# Event-Driven Data Processing Pipeline on AWS

This project implements a serverless, event-driven data processing pipeline using AWS services.

## Key Features
- Event-driven architecture using Amazon EventBridge
- Serverless data processing with AWS Lambda
- Scalable storage using Amazon S3
- Automated daily summary reporting
- Infrastructure provisioned using Terraform Cloud (no local installation)

## Technologies
- AWS Lambda
- Amazon EventBridge
- Amazon S3
- Terraform Cloud
- GitHub

## Deployment
Infrastructure is deployed via Terraform Cloud using a version-controlled workflow.
